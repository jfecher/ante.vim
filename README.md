# ante.vim

Vim plugin providing file recognition and syntax highlighting for the Ante programming language.

Also provides a small colorscheme.

## Installation

Using [Vim-Plug](https://github.com/junegunn/vim-plug), add the following to your `init.vim`:

```vim
Plug 'jfecher/ante.vim'
```

Using [Pathogen](https://github.com/tpope/vim-pathogen):

```
git clone https://github.com/jfecher/ante.vim ~/.vim/bundle/ante.vim
```
